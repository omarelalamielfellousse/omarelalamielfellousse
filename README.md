# 🚀 Project

## 🔐 Phishing Email Detection (NLP + Machine Learning)

👉 [View Project](https://github.com/omarelalamielfellousse/phishing-detection-nlp)

This project focuses on detecting whether an email is **Legitimate** or **Phishing/Spam** using a complete NLP and machine learning pipeline.

### Key Highlights

- Built an end-to-end email classification system
- Combined multiple datasets for better generalization
- Used:
  - word-level TF-IDF
  - character-level TF-IDF
  - handcrafted text features
- Compared several models (Logistic Regression, Naive Bayes, LinearSVC)
- Achieved:
  - **93% accuracy**
  - **0.81 F1-score**
- Developed an interactive **Streamlit interface** for real-time predictions
